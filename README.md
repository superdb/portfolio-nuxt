# Christian Mauro's Portfolio

> Un portafolio moderno, elegante y profesional creado con Vue 3 + Vite + Tailwind CSS

## 🎯 Características

- **Diseño Premium** - Dark theme estilo SaaS con gradientes modernos
- **Fully Responsive** - Se adapta perfectamente a cualquier dispositivo
- **Animaciones Fluidas** - Transiciones y animaciones suaves
- **Performance** - Optimizado con Vite para máxima velocidad
- **SEO Friendly** - Metadata y estructura semántica
- **Componentes Reutilizables** - Arquitectura limpia y escalable

## 📋 Secciones

- **Hero** - Bienvenida impactante con efecto de blobs animados
- **Proyectos** - Showcase de proyectos destacados con hover effects
- **Skills** - Barra de progreso de habilidades técnicas
- **Contacto** - Formulario de contacto funcional
- **Footer** - Enlaces rápidos y redes sociales

## 🚀 Tecnologías

- **Vue 3** - Framework JavaScript reactivo
- **Vite** - Build tool moderno y rápido
- **Tailwind CSS** - Utilidades CSS para diseño
- **JavaScript ES6+** - Características modernas de JavaScript

## 📦 Instalación

1. Clonar el repositorio:
```bash
git clone https://github.com/superdb/superdb-portfolio.git
cd superdb-portfolio
```

2. Instalar dependencias:
```bash
npm install
```

3. Ejecutar el servidor de desarrollo:
```bash
npm run dev
```

4. Abrir en el navegador:
```
http://localhost:5173
```

## 🏗️ Build para Producción

```bash
npm run build
```

El proyecto compilado estará en la carpeta `dist/`

## 📁 Estructura del Proyecto

```
src/
├── components/
│   ├── Navbar.vue       # Barra de navegación
│   ├── Hero.vue         # Sección principal
│   ├── Projects.vue     # Proyectos destacados
│   ├── Skills.vue       # Habilidades técnicas
│   ├── Contact.vue      # Formulario de contacto
│   └── Footer.vue       # Pie de página
├── App.vue              # Componente raíz
├── main.js              # Punto de entrada
└── style.css            # Estilos globales

```

## ✨ Personalización

Para personalizar el portafolio:

1. **Datos Personales** - Edita la sección Hero en `Hero.vue`
2. **Proyectos** - Modifica el array de proyectos en `Projects.vue`
3. **Skills** - Actualiza tus habilidades en `Skills.vue`
4. **Colores** - Personaliza en `tailwind.config.js`

## 📱 Responsive Design

El portafolio está optimizado para:
- 📱 Móvil (320px+)
- 📱 Tablet (768px+)
- 💻 Desktop (1024px+)
